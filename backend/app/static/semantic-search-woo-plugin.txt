Semantic Search for WooCommerce Plugin
========================================

This is a placeholder for the actual plugin ZIP file.

To complete the implementation:
1. Create the actual WordPress plugin ZIP file from the wordpress/wp-content/plugins/semantic-search-woo/ directory
2. Replace this placeholder with the real ZIP file
3. Update the download link in the onboarding page

For now, users can manually download the plugin files from:
c:\xampp\htdocs\wordpress\wp-content\plugins\semantic-search-woo\

Installation Instructions:
1. Go to WordPress Admin → Plugins → Add New
2. Click "Upload Plugin"
3. Select the actual plugin ZIP file
4. Install and activate
5. Enter your license key in the plugin settings
